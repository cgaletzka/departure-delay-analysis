# 2025 Pre-departure delay analysis

This repository contains sql queries, results, and a Tableau visualization for pre-departure delays at European airports in 2025.

## Datasets

The database is made up of four relational tables:

- Delay analyses are based on open data provided by EUROCONTROL: https://ansperformance.eu/csv/#allpre-csv
- Geographical information of each airport were taken from kaggle: https://www.kaggle.com/datasets/samvelkoch/global-airports-iata-icao-timezone-geo
- Regional and runway information are provided in the data folder.
